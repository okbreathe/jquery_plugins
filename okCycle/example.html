okCycle

Tiny, modular, flexible slideshow

Demonstration

Fading

Stop the autoplay by hovering the slideshow 

          
$("#example-1").okCycle({
  autoplay: true,
  effect:'fade', 
  ui: ['caption','navigation','pagination','currentPage']
});

Scrolling

Try using your mousewheel

            
$("#example-2").okCycle({
  effect:'scroll',
  ui: ['caption','navigation','pagination','currentPage', 'mouseWheel']
});

The normal behavior is to use the elements data-caption attribute to provide captions. However, if the data-caption is prefixed with an octothorpe (#), it will be treated as a selector and the innerHTML for the corresponding element will be used.

Sliding

Image loading is deferred until scrolled to (You might have troubling seeing this if you're viewing this locally) 

          
$("#example-3").okCycle({
  effect:'slide', 
  loadOnShow:true,
  ui: ['caption','navigation','pagination','currentPage']
});
